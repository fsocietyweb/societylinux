Getting Started

Society Linux runs as a lightweight layer. Ensure you have bash and curl installed on your host system before running the environment.

How SLR Works

The Society Linux Repository downloads custom-built shell configurations and standalone tools directly into your environment. Because it interacts with system paths, executing installations requires root privileges.

# Install a package from the SLR database
sudo ./societylinux.sh --syspack {package_name}

# Remove an installed package
sudo ./societylinux.sh --uninstall {package_name}
