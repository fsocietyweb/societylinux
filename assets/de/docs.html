Getting Started

Society Linux läuft als schlanke Schicht. Stellen Sie sicher, dass bash und curl auf Ihrem Host-System installiert sind, bevor Sie die Umgebung starten.

How SLR Works

Das Society Linux Repository lädt maßgeschneiderte Shell-Konfigurationen und eigenständige Tools direkt in Ihre Umgebung herunter. Da es mit Systempfaden interagiert, sind für die Durchführung von Installationen Root-Rechte erforderlich.

# Installiere ein Package vom SLRDB (Society Linux Repository Data Base)
sudo ./societylinux.sh --syspack {package_name}

# Entferne ein Installiertes Package
sudo ./societylinux.sh --uninstall {package_name}
